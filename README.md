## Requisitos

- Node.js v14.0 o superior
- NPM v6.0 o superior
- Express.js

## Instalación

Si desea clonar el repositorio(aunque no es necesario ya que la app ya esta desplegada):

```bash
git clone git@github.com:SnapDr4gon/videoClub.git
cd videoClub
npm install
```
